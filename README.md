
# Prototipo – Prueba Técnica Cabify

Este repositorio contiene el prototipo desarrollado para dar respuesta a la prueba técnica solicitada por Cabify. El objetivo del ejercicio era diseñar y construir un módulo interactivo que permitiera a un agente aprender un nuevo procedimiento operativo en menos de 10 minutos, practicarlo en un entorno controlado, validar su nivel de comprensión y recibir retroalimentación inmediata y personalizada.

El prototipo fue implementado utilizando **React**, aprovechando componentes reutilizables, manejo de estado mediante hooks y un diseño responsive alineado con la identidad visual de la marca. Incluye una simulación de caso real, una pregunta interactiva, validación automática de respuestas y un sistema de feedback contextual basado en el desempeño del usuario.

Este proyecto se enfoca en demostrar la lógica, experiencia de usuario y arquitectura propuesta para cumplir con los requisitos técnicos y funcionales establecidos en la prueba.
